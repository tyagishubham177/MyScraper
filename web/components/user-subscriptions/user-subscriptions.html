Subscription Controls
  • ➕ Add products from the available product list
  • 🔎 Use search to filter products
  • 💬 Reach out to admin from the settings pane to add products
  • ❌ Remove a subscription with the X button
  • 🔗 Links open the product page
  • ⏯ Pause or resume your subscriptions when you don't want to be notified
  • ⏰ Set start/end times to set a window in which you want email notifications
